README.md
Objective

The objective of this project is to design and implement a complete software system for E-Services in Pakistan, demonstrating proper software engineering principles including requirements modeling, UML design, and coding practices.

How to Run the Code

Navigate to the 8_Code/src directory.

Compile the code using a C++ compiler:

g++ *.cpp -o EServicesApp

Run the executable:

./EServicesApp
